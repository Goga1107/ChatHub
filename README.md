# SignalR MVC Chat Demo

This is a simple **ASP.NET MVC** web application that demonstrates real-time communication using **SignalR**.

The application contains:
- One simple **User model**
- A **Chat Hub** implemented with SignalR
- Real-time messaging between users opened in different browsers or tabs

The goal of this project is to demonstrate how SignalR works in an MVC application in a simple and understandable way.
